Privacy Policy

Effective Date: June 22, 2026

Last Updated: June 22, 2026

This Privacy Policy applies to the RepeatSegment desktop application ("Application", "App") developed by Sergey Gusev ("Developer", "we", "us"). This document explains what data the Application accesses, how it is used, and what third-party services are involved.

1. Data We Access

The Application processes the following types of data locally on your device:

  • Audio files (.mp3, .wav) — selected by the user for playback, segmentation, and transcription.
  • Microphone input — recorded audio within the Anki Card window for creating custom pronunciations. Recording only happens when the user explicitly presses the "🎤 Rec" button.
  • Text selected by the user — words or phrases highlighted in the transcription panel for translation lookup.

2. Data Sent to Third-Party Services

At the user's explicit request (button press), the Application sends data to the following external APIs:

  • Deepgram (deepgram.com) — audio chunks for speech-to-text transcription. Privacy: deepgram.com/privacy
  • AssemblyAI (assemblyai.com) — audio chunks for speech-to-text transcription. Privacy: assemblyai.com/legal/privacy-policy
  • Yandex Translate (translate.api.cloud.yandex.net) — selected text for English→Russian translation. Privacy: yandex.ru/legal/confidential
  • Google Translate (translate.googleapis.com) — selected text for English→Russian translation (fallback). Privacy: policies.google.com/privacy
  • Deepgram TTS (api.deepgram.com) — single words for text-to-speech synthesis. Privacy: deepgram.com/privacy
  • Google TTS (translate.google.com) — single words for text-to-speech synthesis. Privacy: policies.google.com/privacy
  • Google Images (google.com) — search queries for image lookup. Privacy: policies.google.com/privacy
  • Yandex Images (yandex.ru) — search queries for image lookup (alternative). Privacy: yandex.ru/legal/confidential
  • Dictionary API (api.dictionaryapi.dev) — single words for IPA transcription lookup. No privacy policy; free service.
  • Wiktionary (en.wiktionary.org) — single words for IPA transcription fallback. Privacy: wikimediafoundation.org/privacy

All API keys required for these services are stored locally in a configuration file on the user's device and are never transmitted to the Developer.

3. Data We Do NOT Collect

  • We do not collect, store, or transmit any personal identification information (name, email, IP address, location).
  • We do not use analytics, telemetry, crash reporting, or tracking of any kind.
  • We do not display advertisements.
  • We do not upload your audio files, transcripts, or translations to any server controlled by the Developer.
  • All configuration, cached transcripts, Anki deck data, and downloaded images are stored exclusively in the local AppData folder (%APPDATA%\RepeatSegment).

4. Permissions

The Application requires the following Windows capabilities:

  • Internet (Client) — to communicate with third-party APIs listed in Section 2.
  • Microphone — for audio recording in the Anki Card window. Recording is initiated only by explicit user action.

5. Children's Privacy

The Application is not directed at children under 13. We do not knowingly collect data from children.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated effective date.

7. Contact

If you have questions about this Privacy Policy, you can contact us via the GitHub repository: github.com/Gusev-Sergey/RepeatSegment

Политика конфиденциальности

Дата вступления в силу: 22 июня 2026 г.

Последнее обновление: 22 июня 2026 г.

Настоящая Политика конфиденциальности относится к десктопному приложению RepeatSegment («Приложение»), разработанному Сергеем Гусевым («Разработчик», «мы»). Этот документ объясняет, к каким данным Приложение получает доступ, как они используются и какие сторонние сервисы задействованы.

1. Данные, к которым мы получаем доступ

Приложение обрабатывает следующие типы данных локально на вашем устройстве:

  • Аудиофайлы (.mp3, .wav) — выбираются пользователем для воспроизведения, сегментации и транскрибации.
  • Ввод с микрофона — запись аудио в окне Anki Card для создания пользовательских произношений. Запись происходит только при явном нажатии кнопки «🎤 Rec».
  • Текст, выделенный пользователем — слова или фразы, выделенные в панели транскрипции для перевода.

2. Данные, отправляемые сторонним сервисам

По явному запросу пользователя (нажатие кнопки) Приложение отправляет данные следующим внешним API:

  • Deepgram (deepgram.com) — аудио-фрагменты для распознавания речи. Конфиденциальность: deepgram.com/privacy
  • AssemblyAI (assemblyai.com) — аудио-фрагменты для распознавания речи. Конфиденциальность: assemblyai.com/legal/privacy-policy
  • Yandex Translate (translate.api.cloud.yandex.net) — выделенный текст для перевода EN→RU. Конфиденциальность: yandex.ru/legal/confidential
  • Google Translate (translate.googleapis.com) — выделенный текст для перевода EN→RU (запасной). Конфиденциальность: policies.google.com/privacy
  • Deepgram TTS (api.deepgram.com) — отдельные слова для синтеза речи. Конфиденциальность: deepgram.com/privacy
  • Google TTS (translate.google.com) — отдельные слова для синтеза речи. Конфиденциальность: policies.google.com/privacy
  • Google Images (google.com) — поисковые запросы для подбора картинок. Конфиденциальность: policies.google.com/privacy
  • Yandex Images (yandex.ru) — поисковые запросы для подбора картинок (альтернатива). Конфиденциальность: yandex.ru/legal/confidential
  • Dictionary API (api.dictionaryapi.dev) — отдельные слова для получения IPA-транскрипции. Бесплатный сервис без политики конфиденциальности.
  • Wiktionary (en.wiktionary.org) — отдельные слова для IPA-транскрипции (запасной). Конфиденциальность: wikimediafoundation.org/privacy

Все API-ключи, необходимые для этих сервисов, хранятся локально в конфигурационном файле на устройстве пользователя и никогда не передаются Разработчику.

3. Данные, которые мы НЕ собираем

  • Мы не собираем, не храним и не передаём никакую личную информацию (имя, email, IP-адрес, местоположение).
  • Мы не используем аналитику, телеметрию, отчёты об ошибках или какое-либо отслеживание.
  • Мы не показываем рекламу.
  • Мы не загружаем ваши аудиофайлы, транскрипты или переводы на серверы, контролируемые Разработчиком.
  • Все настройки, кэшированные транскрипты, данные колод Anki и загруженные изображения хранятся исключительно в локальной папке AppData (%APPDATA%\RepeatSegment).

4. Разрешения

Приложению требуются следующие разрешения Windows:

  • Интернет (Клиент) — для связи со сторонними API, перечисленными в разделе 2.
  • Микрофон — для записи аудио в окне Anki Card. Запись инициируется только явным действием пользователя.

5. Конфиденциальность детей

Приложение не предназначено для детей младше 13 лет. Мы сознательно не собираем данные детей.

6. Изменения в политике

Мы можем время от времени обновлять эту Политику конфиденциальности. Изменения будут опубликованы на этой странице с обновлённой датой.

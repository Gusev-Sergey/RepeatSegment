Privacy Policy

Application: RepeatSegment (Android)

Last updated: July 23, 2026

This Privacy Policy describes what data is processed when you use the mobile application RepeatSegment (the "App") and how that data is used.

The developer of the App (the "Developer") is an independent developer, Sergey Gusev, contact: astrorum_arbor@outlook.com.

1. Key principle

The Developer does not collect, store, or share any personal data of the users. The App has no backend servers of its own and no account system. All audio files and user-generated materials (Anki cards, waveform caches) are stored only on the user's device.

2. Data processed locally

The App processes the following data exclusively on the user's device:

  • Audio files opened via the system file picker.
  • Decoding caches (.samples, .waveform) — stored to speed up reloading.
  • User settings (API keys, transcription/translation languages, playback speed) — stored in the Android system storage (SharedPreferences).
  • Anki cards created by the user — stored locally in .apkg format until export.
  • Error log for crash diagnostics — stored locally, never sent to the Developer.

3. Data sent to third-party services

The App works on a BYOK (Bring Your Own Key) model. To use transcription and translation features, the user enters personal API keys in the App's settings. At the user's request, the App sends data directly to the chosen service:

  • Deepgram (US) / AssemblyAI (US) / Yandex (RU) — audio fragments for transcription.
  • Yandex Translate (RU) — text strings for translation.

The Developer has no access to the transmitted data. The user can remove the API key at any time to stop data transmission.

4. In-app purchases

The App may offer a one-time purchase, "RepeatSegment Full Version", which is processed through the YooKassa payment service (ООО "Yumoney"). The payment is collected in favor of the Developer — an individual, payer of tax on professional income (self-employed).

To bind the purchased version to a specific device, the App passes a unique device identifier (Device Code) to the payment creation service. This identifier is not personal data and is used solely to activate the Full Version. Payment data (card number, etc.) is processed only on the YooKassa side and is not shared with the Developer. Processing is governed by the YooKassa privacy policy: yookassa.ru/security.

5. Android permissions

The App requests the following permissions:

  • INTERNET, ACCESS_NETWORK_STATE — to send audio to transcription services, receive responses, create a payment and check the payment status through YooKassa, and to check network availability.
  • FOREGROUND_SERVICE, FOREGROUND_SERVICE_MEDIA_PLAYBACK — to play audio in the background.
  • POST_NOTIFICATIONS — to display a media notification with player controls.
  • WAKE_LOCK — to prevent playback interruptions when the device is idle.
  • READ_MEDIA_AUDIO (Android 13+) / READ_EXTERNAL_STORAGE (Android 12 and below) — to read the audio file selected by the user.
  • RECORD_AUDIO — only for recording the user's voice when creating Anki cards. The recording is stored locally and is not sent anywhere.

The App does not request access to contacts, camera, location, or other sensitive resources.

6. Cookies and analytics

The App does not use cookies, analytics systems, advertising SDKs, or trackers.

7. Children

The App is not directed at children under 13 and does not collect any data from them.

8. Contact

For any questions regarding this Privacy Policy, contact: astrorum_arbor@outlook.com

Политика конфиденциальности

Приложение: RepeatSegment (Android)

Дата последнего обновления: 23 июля 2026 г.

Настоящая Политика конфиденциальности описывает, какие данные обрабатываются при использовании мобильного приложения RepeatSegment (далее — «Приложение») и как эти данные используются.

Разработчик приложения (далее — «Разработчик») — независимый разработчик Сергей Гусев, контактный адрес: astrorum_arbor@outlook.com.

1. Ключевой принцип

Разработчик не собирает, не хранит и не передаёт третьим лицам какие-либо персональные данные пользователей. У Приложения нет собственных серверов и нет системы учётных записей. Все аудиофайлы и созданные пользователем материалы (карточки Anki, кэши волновых форм) хранятся только на устройстве пользователя.

2. Какие данные Приложение обрабатывает локально

Приложение обрабатывает следующие данные исключительно на устройстве пользователя:

  • Аудиофайлы, открываемые через системный диалог выбора файлов (Storage Access Framework).
  • Кэши декодирования (.samples, .waveform) — сохраняются рядом с аудиофайлом или во внутренней директории Приложения для ускорения повторной загрузки.
  • Пользовательские настройки (API-ключи, выбранные языки транскрибации и перевода, скорость воспроизведения) — хранятся в системном хранилище Android (SharedPreferences).
  • Карточки Anki, созданные пользователем, — сохраняются локально в формате .apkg до их экспорта.
  • Журнал ошибок для диагностики сбоев — хранится локально, не отправляется на сервер Разработчика.

3. Передача данных сторонним сервисам

Приложение работает по модели BYOK (Bring Your Own Key). Для функций транскрибации и перевода пользователь самостоятельно указывает персональные API-ключи в настройках Приложения. По запросу пользователя Приложение отправляет данные напрямую в выбранный сервис:

  • Deepgram (США) / AssemblyAI (США) / Yandex (РФ) — аудиофрагмент для транскрибации.
  • Yandex Translate (РФ) — текстовая строка для перевода.

Разработчик Приложения не имеет доступа к передаваемым данным. Пользователь может в любой момент удалить API-ключ из настроек, после чего передача данных прекратится.

4. Покупки внутри Приложения

Приложение может предлагать разовую покупку «Полная версия RepeatSegment», которая обрабатывается через платёжный сервис ЮKassa (ООО «ЮМани»). Оплата совершается в пользу Разработчика — физического лица, плательщика налога на профессиональный доход (самозанятого).

Для привязки оплаченной версии к конкретному устройству Приложение передаёт сервису создания платежа уникальный идентификатор устройства (Device Code). Этот идентификатор не является персональными данными и используется исключительно для активации Полной версии. Платёжные данные (номер карты и т. п.) обрабатываются только на стороне ЮKassa и Разработчику не передаются. Обработка регулируется политикой конфиденциальности ЮKassa: yookassa.ru/security.

5. Разрешения Android

Приложение запрашивает следующие разрешения:

  • INTERNET, ACCESS_NETWORK_STATE — для передачи аудио на сервисы транскрибации, получения ответа, создания платежа и проверки статуса оплаты через ЮKassa, а также для проверки наличия подключения перед сетевыми вызовами.
  • FOREGROUND_SERVICE, FOREGROUND_SERVICE_MEDIA_PLAYBACK — для воспроизведения аудио в фоне (при выключенном экране).
  • POST_NOTIFICATIONS — для показа уведомления с элементами управления плеером во время воспроизведения.
  • WAKE_LOCK — чтобы предотвратить прерывания воспроизведения в спящем режиме устройства.
  • READ_MEDIA_AUDIO (Android 13+) / READ_EXTERNAL_STORAGE (Android 12 и ниже) — для чтения выбранного пользователем аудиофайла.
  • RECORD_AUDIO — только для функции записи голоса пользователя при создании карточек Anki. Запись сохраняется локально в карточку и никуда не отправляется.

Приложение не запрашивает доступ к контактам, камере, геолокации и другим чувствительным ресурсам.

6. Файлы cookie и аналитика

Приложение не использует файлы cookie, системы аналитики (Google Analytics, AppMetrica и т. п.), рекламные SDK или трекеры.

7. Дети

Приложение не ориентировано специально на детей младше 13 лет и не собирает о них никаких данных.

8. Контакты

По всем вопросам, связанным с конфиденциальностью, обращайтесь по адресу: astrorum_arbor@outlook.com
